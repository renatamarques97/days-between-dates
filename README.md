## Days between Dates

### Ruby version
```
2.7.1
```

### Rails version
```
6.0.3
```

### Configuration
```shell
bundle install
yarn install
```

### How to run the test suite
```shell
bundle exec rspec
```

### Run the server
```shell
bundle exec rails server
```

## Documentation
The Project has two inputs: `start_date` and `end_date`, and one output:
the difference between these two dates, which is measured in days and hours.
